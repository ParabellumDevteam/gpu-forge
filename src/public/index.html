HOW A MACHINE BUYS

Three HTTP requests, no account anywhere. Async jobs (video) return a job id immediately; poll /v1/results/{id} for the artifact.

STEP 01

Quote

Call any job endpoint. The forge answers 402 with the exact amount and address.

curl -X POST https://forge.parabellum.tech\ /v1/jobs/llm.generate

STEP 02

Pay

Transfer the quoted USDC from any wallet — yours or your agent's.

→ 0.05 USDC to payTo address

Base (eip155:8453) · Polygon (eip155:137)
or sign an x402 authorization header

STEP 03

Collect

Retry the same request with the transaction hash. Each hash is single-use.

curl -X POST …/v1/jobs/llm.generate \ -H "X-Payment-Tx: 0x…" \ -d '{"prompt":"…"}'
