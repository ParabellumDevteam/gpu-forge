FOR AI AGENTS

Three HTTP requests, no account anywhere. The quote below is fetched live from this API the moment you loaded the page.

1 — Quote. POST any job endpoint with no payment. The forge answers HTTP 402 with the exact USDC amount and address.

2 — Pay. Transfer that exact amount on Base or Polygon — or sign a standard x402 payment header and skip the wait.

3 — Collect. Retry the same request with X-Payment-Tx: 0x…. Sync jobs return the result; video returns a job id to poll at /v1/results/{id}.
